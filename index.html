Sobre a grade do material

HTML

15 tópicos

É uma linguagem de marcação, onde marcamos os elementos para definir quais informações a página vai exibir na web.

CSS

16 tópicos

É uma linguagem de folha de estilo composta por “camadas”, criado com o propósito de estilizar as páginas HTML.

JavaScript

14 tópicos

É uma linguagem de programação que permite a você implementar elementos dinâmicos em páginas web.

Sass

9 tópicos

É um pré-processador CSS que adiciona alguns recursos que não estão disponíveis de forma nativa.

Bootstrap

8 tópicos

É um framework CSS que fornece componentes (como menu, rodapé, card, modal) já prontos para serem utilizados.

React

10 tópicos

É uma biblioteca JavaScript com foco em criar interfaces de usuário de forma componetizada e performática.

React Router

6 tópicos

É uma biblioteca padrão para que você consiga fazer o roteamento das páginas da sua aplicação em React de forma dinâmica.

TypeScript

7 tópicos

É um superset do JavaScript que incluir recursos que não estão presentes na linguagem de forma nativa, além de torná-la estática.

styled-components

7 tópicos

É uma biblioteca que utiliza o conceito de CSS-in-JS, ou seja, que nos permite escrever CSS dentro do JS.

Tailwind CSS

7 tópicos

É um framework CSS que nos fornece classes utilitárias com o propósito de estilizar as páginas.

React Hook Form

8 tópicos

É uma biblioteca que ajuda você a validar formulários no React em poucas linhas de códigos.

Radix UI

6 tópicos

É uma biblioteca de componentes sem estilos, com foco em acessibilidade, personalização e desenvolvimento.

Storybook

6 tópicos

É uma ferramenta que ajuda a documentar variações de cenários diferentes de um componente antes de integrá-lo na aplicação.

Cypress

3 tópicos

É um framework que possibilita a escrita de testes automatizados de forma rápida e confiável, tudo em JavaScript.

Next.js

7 tópicos

É um framework React criado pela Vercel que auxiliar na renderização do lado do servidor e na geração de sites estáticos.

Bônus

5 tópicos

Neste módulo, exploramos as mais recentes tecnologias utilizadas no mercado do desenvolvimento frontend.
